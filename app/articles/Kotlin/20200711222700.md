## 变量、常量与只读

### var 与 val 
两者声明的变量，最本质的区别是：val 不能有 setter

```
class Hello {
    var str: String? = null
        get() {
            return field + "hello"
        }
        set(value) {
            field =  value + "hello"
        }
        
    val str2: String? = null
        get() {
            return field + "hello"
        }
        // 报错 不能有 setter
        set(value) {
            field =  value + "hello"
        }
}
```

注意: val 声明的变量不等于常量

### 编译时常量 const val

+ const 只能修饰 object 的属性，或 top-level 变量
+ const 变量的值必须在编译期间确定下来，所以它的类型只能是 String 或基本类型

```
const val a = 0
```
