## React 的生命周期函数

挂载
1. 调用 `ReactDOM.render`；
2. 进入 `Render 阶段`；
3. 采用 `深度优先遍历` 创建 `fiber 树`；
4. 进入 `Commit 阶段`；


更新
1. 调用 `this.setState`；
2. 进入 `Render 阶段`；
3. 采用 `深度优先遍历` 创建 `fiber 树`；
4. `reconcile 算法` 标记变化；
5. 进入 `Commit 阶段`；
6. 执行 `4` 中 `变化` 对应的 `视图操作`；
